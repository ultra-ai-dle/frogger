# QA: Example Gallery

> 마지막 업데이트: 2026-04-12
> 대상 파일: src/features/gallery/ExampleGallery.tsx, src/features/gallery/ExampleCard.tsx, src/features/gallery/useGallery.ts, src/data/examples.ts, app/page.tsx

## 정상 동작

- [ ] **갤러리 열기** — 헤더 우측의 파일 아이콘 버튼을 클릭하면 → 갤러리 모달이 화면 중앙에 오버레이로 열린다
- [ ] **카테고리 탐색** — 왼쪽 사이드바에서 "그래프"를 클릭하면 → 카드 그리드가 그래프 카테고리 예제(BFS, DFS, Dijkstra, Union-Find)로 전환된다
- [ ] **모든 카테고리 확인** — 6개 카테고리(정렬, 탐색, 자료구조, 그래프, DP, 재귀)를 순서대로 클릭하면 → 각 카테고리마다 해당 예제 카드들이 표시된다
- [ ] **예제 선택 (코드 비어있을 때)** — 에디터가 비어있는 상태에서 예제 카드를 클릭하면 → 에디터에 코드가 로드되고 stdin이 입력되며 모달이 닫힌다
- [ ] **예제 선택 (Python)** — Python 예제(Bubble Sort 등)를 선택하면 → 에디터에 Python 코드가 로드되고 언어가 Python으로 설정된다
- [ ] **예제 선택 (JavaScript)** — 언어 토글에서 JS를 선택 후 카드를 클릭하면 → 에디터에 JavaScript 코드가 로드되고 언어가 JavaScript로 설정된다
- [ ] **선택 후 실행** — 예제를 선택한 뒤 실행 버튼을 누르면 → 해당 알고리즘이 정상 실행되고 시각화가 표시된다

## 엣지 케이스

- [ ] **덮어쓰기 확인** — 에디터에 코드가 있는 상태에서 예제 카드를 클릭하면 → "현재 코드를 덮어쓸까요?" 인라인 다이얼로그가 모달 내부에 표시된다
- [ ] **덮어쓰기 확인 → 확인** — 인라인 다이얼로그에서 "확인"을 클릭하면 → 예제 코드로 교체되고 모달이 닫힌다
- [ ] **덮어쓰기 확인 → 취소** — 인라인 다이얼로그에서 "취소"를 클릭하면 → 카드 그리드로 돌아가고 기존 코드는 유지된다
- [ ] **카테고리 변경 시 확인 초기화** — 덮어쓰기 확인 상태에서 다른 카테고리를 클릭하면 → 확인 다이얼로그가 사라지고 해당 카테고리의 카드 그리드가 표시된다
- [ ] **언어 토글 클릭** — 카드 내 [Py]/[JS] pill을 클릭하면 → pill 선택이 전환되고 카드 자체는 선택되지 않는다 (이벤트 버블링 차단)
- [ ] **단일 variant 카드** — variant가 1개뿐인 예제가 있다면 → 언어 토글 pill이 표시되지 않는다
- [ ] **태그 툴팁** — 예제 카드 위에 마우스를 올리면 → 브라우저 기본 툴팁으로 태그 목록(예: "sorting, array")이 표시된다

## 에러 케이스

- [ ] **ESC 키 — 모달 닫기** — 갤러리 모달이 열린 상태에서 ESC를 누르면 → 모달이 닫힌다
- [ ] **ESC 키 — 확인 다이얼로그 닫기** — 덮어쓰기 확인 다이얼로그가 표시된 상태에서 ESC를 누르면 → 확인 다이얼로그만 닫히고 모달은 유지된다
- [ ] **배경 클릭 닫기** — 모달 바깥 검은 오버레이 영역을 클릭하면 → 모달이 닫힌다
- [ ] **모달 내부 클릭** — 모달 내부(카드, 사이드바 등)를 클릭하면 → 모달이 닫히지 않는다 (이벤트 전파 차단)

## UI 상태

- [ ] **갤러리 버튼 위치** — 헤더 우측에 갤러리 아이콘 버튼이 투어 아이콘 왼쪽에 위치한다
- [ ] **갤러리 버튼 hover** — 갤러리 버튼에 마우스를 올리면 → "예제 갤러리" 툴팁이 표시된다
- [ ] **카테고리 활성 상태** — 선택된 카테고리는 흰색 텍스트 + 어두운 배경으로 강조되고, 나머지는 회색이다
- [ ] **난이도 뱃지** — Easy 예제는 초록 뱃지, Medium 예제는 노란 뱃지로 표시된다
- [ ] **언어 pill 활성 상태** — 선택된 언어 pill은 파란 테두리 + 파란 텍스트로 강조되고, 나머지는 회색이다
- [ ] **카드 hover** — 예제 카드에 마우스를 올리면 → 테두리가 파란색으로 변하고 배경이 밝아진다
- [ ] **모달 레이아웃** — 모달이 열려도 페이지 스크롤이 발생하지 않는다 (100vh 고정 레이아웃 유지)
- [ ] **모달 최대 높이** — 모달은 화면 높이의 80%를 넘지 않고, 카드가 많으면 카드 영역만 스크롤된다
- [ ] **푸터 텍스트** — 모달 하단에 "Python · JavaScript 지원 | 선택하면 에디터에 로드" 텍스트가 표시된다
